Diploma Final Year Project

Github

My Diploma Final Year Porject is a human resources system. On this system, admin can create the required positions and collect the resumes of job seekers. Managers can rate employees' performance and send emails to employees.

I joined a simple email system, and employees can communicate by email. Employees can also apply for sick leave or vacation through the system. The special feature of this system is that I use face recognition to check in. When employees join the company, they will collect head photos and let the training library identify them.

System is full build by Laravel. The Face Recognition Library is using the JS Library. The system still has a lot of room for improvement. The code aspect is the first time that I finished it by myself, so there are many bad places and I can modify them.
